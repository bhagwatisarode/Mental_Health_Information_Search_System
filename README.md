# Mental Health Information Search System

# Overview
A Mental Health Information Search System is a web-based platform that helps user find curated mental health resource efficiently. It uses TF-IDF scoring to rank documents by relevance and provides a user-friendly interface for searching and exploring mental health topics.

# Features

- Keywords-based Search for mental health topics, services, and reports.
- Highlighting of search term in result.
- Responsive, clean, and intuitive web interface.
- Document meatadata and tags for easier exploration.


# Technologies Used
- HTML, CSS, JavaScript
- Bootstrap 4 for responsive design
- TF-IDF algorithm for ranking search results
- JSON dataset of mental health documents

# Dataset
- Curated dataset of 300 mental health documents from NHS and other reliable sources.
- Each document contains title, summary, URL, category, tags, and metadata.

# HOW to Run
1. Open the project folder.
2. Open 'index.html' in any modern web browser.
3. Enter keywords in the search bar to find relevant mental health resources.
